# Word Inflection

**Word Inflection** is a Visual Studio Code-compatible extension that
cycles and converts word naming cases.  This is a port of
[*string-inflection*](https://github.com/akicho8/string-inflection) for
Emacs.

## What This Extension Does

This extension provides commands to:

- Cycle through language-specific naming conventions
- Convert directly to `snake_case`, `camelCase`, `PascalCase`, `kebab-case`,
  `SCREAMING_SNAKE_CASE`, or `Capital_Snake_Case`
- Transform word parts (subwords) based on VS Code's word-part navigation
- Work with Unicode letters in conversions and detection

All operations are performed **entirely within the editor buffer**.

---

## Security & Privacy

- This extension **does not execute external commands**
- It **does not access the network**
- It **does not collect telemetry or user data**

All behavior is limited to editor text manipulation using the VS Code API.
